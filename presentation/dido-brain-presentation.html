The problem

The team
flies blind.

Work and decisions don't live in tidy status reports. They live scattered inside individuals' tools, where others can't find them, or even know they exist.

Knowledge is siloed & undiscoverableIt isn't lost; it's trapped in someone's Fellow, Cowork or Slack, effectively invisible.
Work is inefficientPeople redo analysis, hunt for context, and decide on partial information.
The solution

One brain for
the whole team.

DiDO ingests the work artifacts scattered across everyone's tools into a time-aware org brain that connects them, and answers natural-language questions, grounded in real sources.

Value prop 01

Connected knowledge

Unify what's scattered across Fellow, Cowork and Slack into one place anyone can reach. Trapped information becomes discoverable.

Value prop 02

Democratized transparency

Anyone can ask a question and get a cited answer from real artifacts, so everyone can function like a leader.

Why this project

Why a shared brain.

A shared team brain is the unlockWith AI, giving everyone leader-level visibility into the whole company is a step-change, not an incremental gain.
Study a brain that already worksWe researched gbrain to learn what a well-thought-through brain actually looks like, so we hold an informed opinion on the right brain and ingestion process for Sierra.
Don't reinvent the hard partLean on production-grade memory infrastructure and strip away the complexity of building our own brain, so our effort goes to solving the problem.

Deep dive · gbrain infrastructure

Git is truth.
Postgres is speed.

Two stores, one rule: durable markdown in git is the source of truth; Supabase is a rebuildable cache built for retrieval.

Source of truth

GitHub markdown repos

Knowledge is written as plain-text markdown. Every overnight change is a git diff you can review. Transparency is native to the store, not bolted on.

Query engine

Supabase · Postgres + pgvector

gbrain sync imports the repo, chunks each page, embeds it (text-embedding-3-large, 1536d) and serves hybrid vector + keyword retrieval.

Deep dive · The dream cycle

The brain sleeps to stay sharp.

A nightly cron runs gbrain's dream cycle, a stateless batch job that consolidates the day's new knowledge while no one's watching. Inspired by biological sleep.
